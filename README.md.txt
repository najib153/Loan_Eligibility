# Loan Eligibility Prediction

## Overview
This project predicts loan approval using machine learning models. It includes data preprocessing, model training, and evaluation.

## Project Structure

loan-eligibility/
├── data/
│   └── credit.csv
├── src/
│   ├── data_processing.py
│   ├── modeling.py
│   ├── loan_eligibility.py
│   └── app.py
├── requirements.txt
└── README.md

## Setup & Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/Loan_Eligibility_Project.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Loan_Eligibility_Project
   ```
3. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

## Usage
1. Run the main script to process data and train models:
   ```sh
   python main.py
   ```
2. Check logs for model performance and errors.

## Dependencies
- Python 3.x
- Pandas
- Scikit-learn
- NumPy
- Matplotlib
- Seaborn
